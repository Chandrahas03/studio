# **App Name**: Chatterbox AI

## Core Features:

- New Conversation: Start a new chat session with the AI assistant.
- Message Input: User interface for typing and sending messages.
- AI Response: Receive AI-generated responses to user messages using generative AI. The responses will be based on a complex tool which combines multiple sources of knowledge and different reasoning strategies, so that it is not prone to 'hallucination'.
- Chat History: View the history of messages in the current chat session.

## Style Guidelines:

- Primary color: Teal (#008080) for a calming and supportive feel. This color was requested by the user.
- Background color: Very light desaturated teal (#F0FFFF) as requested by the user, to ensure contrast and readability.
- Accent color: Light blue (#ADD8E6) to highlight interactive elements and calls to action.
- Body and headline font: 'Inter', a sans-serif font, will be used for both headings and body text for a clean and modern look. This font will ensure high readability and a contemporary feel throughout the application.
- Use simple, clear icons to represent actions and functions within the chat interface.
- Clean and intuitive layout for easy navigation and readability. Focus on a single column design for the chat interface.
- Subtle animations to indicate loading and new messages.
